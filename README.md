# 安装
## 建议配置源库

```text
composer config -g repo.packagist composer https://mirrors.aliyun.com/composer/
```

## 安装

```text
composer require dtapp/think-bt -vvv
```

## 更新

```text
composer update dtapp/think-bt -vvv
```

## 删除

```text
composer remove dtapp/think-bt -vvv
```
